# Wikist 站点升级日志

本日志记录代码层面的框架升级、数据结构调整与运维注意事项；用户编辑、评论、评分等内容行为仍记录在后台“更新日志”中。

## 2026-07-10 - 启动器与端口自愈

- Windows 启动器新增 Node.js 主版本检查，并依次识别系统 PATH、项目内便携运行时与本机 Codex 运行时。
- 项目根目录新增 `run-wikist-server.cmd`，无需先进入 `tools` 目录即可启动。
- 默认端口 `8899` 保持严格监听，不再自动切换；新增 `run-wikist-server.cmd --restart`，仅在确认该端口响应的是 Wikist 后重启旧实例，以同一端口加载新代码。

## 2026-07-10 - Passport 外部资料

- 用户资料新增个人网站、博客、GitHub、知乎、哔哩哔哩、X 与 Mastodon 链接，并同步展示在账户身份档案和公开用户主页。
- `users.social_links_json` 由迁移自动创建；外部资料仅接受 `http` / `https` 地址，公开接口不会暴露邮箱。
- 当前端已更新而运行中后端尚未重启时，保存外部资料会给出明确的 SweetAlert 重启提示，不再静默显示“已保存”。

## 2026-07-10 - 安装器、账号状态与函数图修复

- 新增 `/install.html` 初始安装向导：可配置站点身份、SQLite 相对路径、协作策略与 SMTP；首次安装可直接写入 `config/site.config.json`，完成后重启加载。
- 现有站点默认锁定安装器；仅在 `WIKIST_INSTALL_MODE=1` 显式维护模式下允许重新生成基础配置，避免公开入口覆盖站点设置。
- `function-plot` 特殊函数改为将 math.js 函数显式注入 function-plot 的表达式作用域，修复 `gamma(x)` 的 `symbol "gamma" is undefined`，并保持标准拖拽缩放渲染链。
- 修复被禁用账号登录出现乱码；改为明确的 SweetAlert 封禁提示。被封禁用户的公开档案、历史贡献仍可查看，并显示协调的封禁状态说明。
- 用户中心重新编排为固定区域网格，安全、翻译、收藏、密码和公开资料在桌面端对齐，移动端顺序展开。
- 后台消息优先级选择控件补齐明暗主题的前景色、背景色和原生下拉选项样式。

## 2026-07-10 - 函数绘图、消息优先级与 Passport 安全

- 函数图像回到 `function-plot` 标准渲染链：特殊函数按需加载 math.js，并使用 `function-plot` 内置采样器完成拖拽、缩放和重绘；`implicit:` 隐函数改为区间图像模式。
- 后台消息群发的优先级控件重做为紧凑表单，urgent 最高优先级支持配置右上角弹窗展示秒数。
- Passport 新增邮箱验证、找回密码、一次性哈希令牌、SMTP 邮件发送配置和公开配置脱敏。
- 用户安全中心新增 TOTP 二次验证开关、安全状态提示和验证邮件重发入口；二次验证密钥加密存储。

## 2026-07-10 · 收藏索引与数学建模插件

- 用户中心新增“我的收藏”独立分页页，保留最近 10 条概览，并以搜索结果样式展示全部收藏词条。
- 新增收藏列表 API，按收藏时间分页返回词条摘要、分类、质量、状态与归档状态。
- 函数图像插件接入 mathjs 按需表达式引擎，支持 `gamma`、`erf`、`zeta`、`sinc`、`besselj`、`besseli` 和 `implicit:` 隐函数采样绘图。
- 新增 JSXGraph 交互几何插件与 Chart.js 数据建模插件；均采用 manifest、后台启停、围栏语法和懒加载客户端模块。

## 2026-07-10 · 翻译工作台、收藏与统一弹窗

- 重构词条翻译工作台：源文默认以渲染视图呈现，可切换 Markdown 源码；译文标题、摘要、正文和进度集中在连续双栏界面中，并完成移动端适配。
- 新增 `page_favorites` 收藏表、词条收藏状态接口与用户收藏列表，重复收藏保持幂等，账户中心最多读取最近 10 条。
- 词条标题区新增收藏按钮，显示全站收藏数；登录用户可收藏或取消收藏，访客会得到登录引导。
- 引入 SweetAlert2，替换原生语言输入框与备份回档确认框，并统一收藏、翻译等操作提示。
- SweetAlert2 11.26.25 固定在本地 `public/assets/vendor`，不依赖运行时 CDN，保留 MIT 许可证并跟随站点明暗主题。

## 2026-07-09 · 语言与审计日志收敛

- 词条语言切换改为阅读模式：`#/page/<slug>?lang=<code>` 直接展示对应译文正文，不再跳入翻译工作台。
- 翻译工作台保留在 `#/translate/<slug>?lang=<code>`，用于生成初稿、编辑和保存译文。
- 翻译语言从固定三种扩展为 BCP-47 风格语言代码，支持 `fr`、`ja`、`de-DE` 等自定义语言。
- 站点设置新增语言列表配置，顶部语言菜单可跟随站点语言扩展。
- 用户中心“翻译社区”改为标签式语言选择，并支持输入自定义语言代码。
- 翻译读取接口返回译文 HTML 与目录，前端阅读页无需重复实现 Markdown 渲染。
- 后台更新日志收敛为内容和站点变更：保留创建、删除、发布、保存、权限、备份回档等关键操作；登录、登出、消息已读和消息删除不再写入审计日志。
- 首页配置继续保留后台可维护能力，包括资讯雷达与全球数学进展 JSON。

## 2026-07-09 · 翻译社区与首页配置

- 新增翻译社区数据表、词条译文表与审计日志表。
- 新增词条翻译入口、自动翻译初稿、翻译度统计与中英/中繁对照工作台。
- 新增后台首页展示内容配置，允许维护首页标题、简介、资讯雷达和全球数学进展。
- 新增评论删除能力，管理员可删除全部评论，用户可删除自己的评论。
- 修复目录锚点点击破坏 hash 路由的问题。
## 2026-07-10 - 消息优先级与函数图交互

- 全站群发加入 low / normal / high / urgent 优先级，urgent 在首次送达时以右上角弹窗呈现。
- 顶部消息按钮改为异步下拉预览，保持一个群发正文与用户状态的轻量存储。
- mathjs 特殊函数和隐函数 SVG 图像支持指针拖动平移，视域会在重绘时保留。
- 收藏列表的取消操作使用独立危险视觉样式与删除图标。
