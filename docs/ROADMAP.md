# Wikist Roadmap

## 0.1 Foundation

- Markdown file articles.
- Revision snapshots.
- Chinese and English search.
- MathJax formula rendering.
- Responsive reading and editing UI.

## 0.2 Knowledge Graph

- Backlinks and missing-page reports.
- Category pages.
- Term aliases and bilingual terminology.
- Page quality dashboard.

## 0.3 Review Workflow

- Draft revisions separated from published pages.
- Reviewer approval.
- Page discussion threads.
- Source and citation validator.

## 0.4 High-Performance Deployment

- Optional SQLite or PostgreSQL storage.
- Static article cache.
- Search index persistence.
- CDN-friendly asset pipeline.

## 1.0 Community Edition

- Account system.
- Role-based permissions.
- Extension API.
- Importers for MediaWiki and Wiki.js exports.
