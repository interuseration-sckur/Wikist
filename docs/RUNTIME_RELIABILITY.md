# 运行可靠性与请求防护

Wikist `0.12.0` 将运行可靠性保持在单机、单 SQLite 文件的轻量边界内：不需要 Redis、Elasticsearch、消息队列或额外守护进程。

## SQLite 并发

Passport 数据库启动后会设置：

- `journal_mode = WAL`：读请求不再因普通写入而整体阻塞。
- `busy_timeout = 8000`：短时写锁会等待，而不是立刻把并发写入变成错误。
- `synchronous = NORMAL`、`wal_autocheckpoint = 1000`、`foreign_keys = ON`：在常规 VPS 与本地部署中平衡吞吐、恢复性和关系完整性。

后台 **运行健康** 会展示 WAL 状态、busy timeout、数据库大小与 `PRAGMA quick_check(1)` 结果。健康检查不会输出账户、会话、访客地址或评论正文。

## 搜索索引恢复

FTS5 是派生索引，Markdown 始终是词条事实来源。若 FTS5 建表、增量写入或查询失败：

1. 当前词条保存继续完成。
2. 搜索自动退回内置轻量索引。
3. **后台 -> 搜索索引** 与 **后台 -> 运行健康** 显示“待修复”。
4. 管理员选择“修复搜索索引”，系统仅重建 `wikist_page_fts` 与索引状态表，再从现有词条回填。

修复不修改 Markdown、修订记录、用户数据或页面权限。

## 备份校验与还原演练

新的 `wikist-site-backup` v2 在每个文件记录 SHA-256，并对整体清单记录校验值。解析与还原会检查：

- gzip / JSON 格式和白名单路径；
- 内容编码、声明大小与 SHA-256；
- 重复路径与用户数据库文件是否匹配本站配置的 SQLite 相对路径。

旧版备份仍可导入，但后台会将其标示为 `legacy`，只执行路径与内容结构检查。

手动下载备份时，运行中的 Passport 数据库会先通过 SQLite `VACUUM INTO` 生成一致性快照，避免只复制 WAL 主文件导致的遗漏。后台 **运行健康 -> 执行还原演练** 会在系统临时目录创建隔离站点、还原一次刚创建的快照并立即删除临时目录；不会写入当前站点。默认只演练内容，可勾选“包含用户数据”一并验证数据库文件。

全量用户数据回档会临时关闭并重新打开当前 SQLite 连接，因此当次请求完成后即可继续使用；仍建议在生产环境选低流量时段执行。

## 脱敏运行指标

指标只保存在当前 Node.js 进程内，重启后自然清空。后台 **运行健康** 提供：

- 请求数量、路由平均耗时、最大耗时和 5xx 数量；
- 搜索平均耗时、轻量搜索缓存命中率、FTS5 / 回退使用次数；
- Hook 与可信客户端模块的失败计数；
- 限流、超大请求体和安装校验拦截次数。

不会记录 IP、账号、邮箱、User-Agent、搜索词、词条正文或错误堆栈。公开 `/api/health` 仅返回存活、数据库可用性和索引就绪摘要；完整状态只对后台角色开放。

## 请求与安装防护

Wikist 内置的是单机内存限流器，参考 `node-rate-limiter-flexible` 的按键窗口与阻断思路，但不额外引入运行依赖。默认策略区分：

- 普通站点访问：240 次 / 分钟；
- 读取 API：120 次 / 分钟；
- 写入操作：48 次 / 分钟；
- 登录、注册、找回密码、人机验证：16 次 / 分钟；
- 安装接口：8 次 / 10 分钟。

超限返回 `429`、`Retry-After` 与标准限流响应头。请求体默认上限为 2 MiB。安装页先向 `/api/install/status` 获取短时校验令牌，配置写入与卸载请求必须携带它；若浏览器提供跨域 `Origin`，其必须与当前站点一致。已完成安装时，既有的安装锁仍优先保护配置。

在 Nginx、Caddy 或 Cloudflare 后面部署时，只有确认代理会清理外部伪造的 `X-Forwarded-For` 并重新设置该头，才在 **后台 -> 运行健康** 勾选“信任反向代理”。否则保持关闭。

内置防护适合单机 CC、误操作和低成本撞库缓解，不能替代 CDN/WAF、反向代理限速、操作系统防火墙或分布式 DDoS 防护。

## 自动检查

```powershell
npm run check:runtime
npm run check:backup
npm run check:search
npm run check:hooks
```

`check:runtime` 会在临时目录验证 WAL、数据库快照、FTS 故障恢复、匿名限流、安装令牌、脱敏指标、插件 Schema 迁移以及服务端模块默认不执行边界。
